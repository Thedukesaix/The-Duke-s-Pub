The Duke's — Pack site (splash + carte images)
================================================

Fichiers :
- index.html        → Page d'accueil (logo centré + redirection vers menu.html)
- index_single.html → Variante "1 seul fichier" (logo intégré en base64) + redirection vers menu.html
- menu.html         → Affiche les 4 pages de la carte (Page1..Page4)
- logo.png, Page1.png, Page2.png, Page3.png, Page4.png

Mise en ligne :
1) Allez sur https://app.netlify.com/drop
2) Glissez tout le dossier
3) Copiez l'URL publique générée
4) Générez un QR code vers l'URL de index.html
